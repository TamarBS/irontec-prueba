# IrontecPrueba

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 12.2.11.

## Steps to run the proyect

1.: Clone the repository on your local or download the proyect
2.: Open the console and find the folder irontec-prueba on your computer
3.: Write ng serve to run the proyect
4.: Open your browser and copy the link 'http://localhost:4200/'

You must see the main page
